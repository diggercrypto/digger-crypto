# DIGGER_CRYPTO — Static site

## Setup & Deploy (GitHub Pages)
1. Replace all occurrences of <YOUR_GITHUB_USERNAME> and <REPO_NAME> in files.
2. Add logo images to /assets.
3. Commit and push to GitHub.
4. In repository Settings > Pages, select branch `main` (or `gh-pages`) and folder `/ (root)`.
5. Wait a few minutes — site will be available at https://<YOUR_GITHUB_USERNAME>.github.io/<REPO_NAME>/

## Notes
- Edit app.js to update contract addresses or API usage.
- Replace GA id in index.html to enable analytics.
- To improve SEO ranking: get backlinks, publish quality content, add structured blog posts and a Press/News page.
